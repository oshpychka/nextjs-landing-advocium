## Встановлення та запуск проєкту

### 1. Клонування репозиторію
git clone https://github.com/oshpychka/nextjs-landing-advocium
cd REPOSITORY

### 2. Встановлення залежностей
npm install

### 3. Запуск у режимі розробки
npm run dev

Проєкт буде доступний за адресою:
http://localhost:3000

### 4. Збірка проєкту
npm run build
